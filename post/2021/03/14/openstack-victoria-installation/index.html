OpenStack Victoria をマニュアルインストールしてみたメモ

これまでの OpenStack シリーズの記事では Ubuntu 20.04 LTS の apt で標準パッケージとなっているリリース名「Ussuri」（リリース日：2020-05-13）のインストールについてメモしていきました

今回は次のリリース「Victoria」（リリース日：2020-10-14）のインストールについてのメモです
（もうあと 1 ヶ月もすれば「Wallaby」がリリースされるタイミングですが…）

といっても内容はとても薄いです

インストール手順

  1. apt を Victoria に対応させる

    OpenStack Victoria for Ubuntu 20.10 and Ubuntu 20.04 LTS – wrestling penguins

    #root user

    add-apt-repository cloud-archive:victoria

  2. あとは Ussuri のときと同じです！

    • OpenStack Docs: OpenStack Installation Guideにまだ Victoria のまとめがないようですが、各プロジェクトの Victoria のインストール手順を確認したところ特に変わったところは見られませんでした
    • 実際OpenStack Ussuriをマニュアルインストールしてみたメモの手順でインストールできることを確認しました
    • とはいっても…
      OpenStack Docs: OpenStack Installation Guideは頻繁に更新されるので、このメモは参考程度に最新版のドキュメントに倣ってインストールした方がよいです
